CSS selectors and specificity

Tag Name Description
.column Class selector Can be reusable & applied to many elemnts. They are denoted with a period before the class.
#first-item CSS id selector Should be unique to style only one element. They are denoted with a hash sign.
h1, h2 Groups of CSS selectors Match multiple selectors to the same CSS rule. Use comma to seperate.
h3.section-heading Chaining selectors Chained so that rule sets apply only to elements that match all criteria.
p CSS type selectors To match all elements of a given type or tag name. They are matched regardless of their nesting level in the HTML.
.calendar-cell CSS class selectors Matches elements based in the contents of their class attribute. To select ir we need a period infront.
